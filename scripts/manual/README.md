# Manual Scripts

These scripts are not part of the supported verification surface. They are retained only for explicit operator/debug use.

Categories:

- `maintenance/`
  One-off operational helpers for import, sync, or template builds.
- `benchmarks/`
  Simple local measurement helpers.

The supported automated checks remain:

- `bun run check`
- `bun run web:typecheck`
- `bun run web:test:ui`
